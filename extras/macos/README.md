# macOS Extras

Optional macOS user-level configs.

| Extra | Target | Purpose |
|---|---|---|
| `macos-disable-navigation-keys` | `~/Library/KeyBindings/DefaultKeyBinding.dict` | Disables macOS navigation shortcuts that conflict with editor keymaps |

The keybinding file maps `Cmd+Ctrl+Left/Right`, `Cmd+Opt+Left/Right`, and
`Cmd+Opt+Ctrl+Left/Right` to `noop:`.
